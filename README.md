# CS 141 HW1

Before starting, create a new directory on your machine for this assignment. Navigate to that directory.

To clone your repository, click the green Code button, copy the web URL, and run:
```bash
git clone <YOUR WEB URL>
```

To commit + push your changes:
```bash
git add mainClass.java
git commit -m "Description of your change"
git push origin main
```

To compile:
```bash
javac mainClass.java
```

To run:
```bash
java mainClass <...arguments...>
```
